+   模式转换
    默认打开普通模式,输入 i 进入编辑模式,按 esc 回到普通模式
    普通模式中输入 : 进入命令行模式
+   命令行模式中输入 wq 保存退出
+   不保存内容强制退出 q!
    未进行编辑 q 即可退出
+   解决鼠标不能用的问题
    修改vim的配置文件 vim ~/.vimrc
    添加 set mouse=i
    ！只有在编辑模式下可用
+   vim 上下左右移动
    左:h 右:j 上:k 下:l
    ！普通模式下才有用
+   .swp 文件
    vim 的交换文件，用于在未保存文件时临时保存文件
    打开一个文件就会产生相应的.swp文件
+   打开报错的处理
    1. 同时打开了一个文件进行编辑
    用 ps aux | grep vim 检查 vim 的使用情况
    2. 不正常退出
    可以通过 .swp 文件恢复
    这时 .swp 仍然存在，用 rm 删除相应的.swp 文件
+   跳转到第 n 行
    跳转到第一行: gg
    跳转到最后一行: G
    跳转到指定行: :n
    ！普通模式下才有用
+   复制、粘贴、剪切一行代码
    复制一行代码: yy (按了之后没有什么具体的响应)
    粘贴: p
    剪切一行代码: dd
    ！普通模式下才有用
+   搜索文件内容
    搜索命令: /
    下一个：n 上一个: N
    ！普通模式下
+   滚动屏幕
    向上滚动: ctrl + u
    向下滚动: ctrl + d
+   撤销: u
    反撤销: ctrl + r
    ！普通模式
+   指令汇总：
    进入编辑模式 i
    进入命令行模式 :
    保存并退出 :qw
    强制退出 :q!
    未修改的退出 :q
    向上移动 k
    向下移动 j
    向左移动 h
    向右移动 l
    向上滚动 ctrl + u
    向下滚动 ctrl + d
    移动到最上面 gg
    移动到最下面 G
    移动到指定行 :n
    剪切 dd
    复制 yy
    粘贴 p
    撤销 u
    反撤销 ctrl + r
    搜索 /
    搜索下一个 n
    搜索上一个 N
+   代码
    1.让鼠标可用
    vim ~/.vimrc
    set mouse=i
    2.查看vim使用情况
    ps aux | grep vim 
    3.删除文件
    rm
