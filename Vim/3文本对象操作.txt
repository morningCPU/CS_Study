文本对象操作
+   {count}{operator}{textobjects}
+   textobjects : 语义化的文本片段
    格式：i/a + 对象
+   常见的对象
    w/W 单词
    s   句子
    p   段落
    {/},[/],</>,/   配对符定义的对象
+   i : 内部
    a : 额外包括周围的空格和配对符

操作符与命令补充
+   gu  转小写
+   gU  转大写
+   g~  翻转大小写
+   ctrl + a    增加数字
+   ctrl + x    减小数字
+   g<ctrl-A>   创建递增序列(要先选中)
+   <   左缩进
+   >   右缩进
+   J(大J)  连接两行
